  4  * Many objects in Node emit events: a <code>net.Server</code> emits an event each time

  7  * You can access this module by doing: <code>require("events");</code>
  8  *
  9  * Typically, event names are represented by a camel-cased string, however,
 10  * there aren't any strict restrictions on that, as any string will be accepted.
 11  *
 12  * Functions can then be attached to objects, to be executed when an event
 13  * is emitted. These functions are called <em>listeners</em>.
 14  */
